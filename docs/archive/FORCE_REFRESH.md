# 强制刷新浏览器缓存

**问题**: 眼睛图标按钮不可见

**原因**: 浏览器缓存了旧版本的 HTML/JavaScript 文件

---

## 🔧 解决方案

### 方法 1: 强制刷新 (推荐)

**Windows/Linux**:
```
Ctrl + F5
```

**macOS**:
```
Cmd + Shift + R
```

---

### 方法 2: 清除缓存

**Chrome/Edge**:
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

**Firefox**:
1. 按 `Ctrl + Shift + Delete`
2. 选择"缓存"
3. 点击"立即清除"

---

### 方法 3: 禁用缓存 (开发模式)

**Chrome/Edge**:
1. 按 `F12` 打开开发者工具
2. 进入"Network"标签
3. 勾选"Disable cache"
4. 保持开发者工具打开状态

---

---

## 📝 已修复的问题

1. ✅ 模态框宽度从 `modal-lg` 改为 `modal-xl`
2. ✅ 添加 input-group CSS 样式
3. ✅ 添加按钮最小宽度 `min-width: 50px`
4. ✅ 添加图标字体大小 `font-size: 1.2rem`
5. ✅ 添加输入框 `min-width: 0` 允许收缩

---

**如果强制刷新后仍然不可见，请检查浏览器控制台是否有 JavaScript 错误。**
