## 📝 使用示例

### 创建动画

```python
from src.animation import Timeline, create_move_animation

# 方法 1: 使用便捷函数
timeline = create_move_animation(
    "object_1",
    start_pos=(0, 0),
    end_pos=(100, 50),
    duration=2.0,
    easing='ease_in_out_quad'
)

# 方法 2: 手动构建
timeline = Timeline(fps=60)
channel = timeline.add_channel("object_1")
channel.animate('x', 0, 100, duration=2.0, easing='ease_in_quad')
channel.animate('y', 0, 50, duration=2.0, easing='linear')

# 生成帧
frames = timeline.get_frames()
```

### Manim 集成

```python
from src.camera.manim_adapter import ManimSceneRenderer

renderer = ManimSceneRenderer()

# 快速演示
output = renderer.quick_demo("demo.mp4")

# 自定义场景
scene = renderer.create_scene("MyScene")
# ... 添加对象和动画
renderer.render(scene, "output.mp4")
```
